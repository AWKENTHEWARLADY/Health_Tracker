HealthFlow is a full-stack health and wellness tracking web application that enables users to securely manage workouts, nutrition, health metrics, and medications.  
The system is designed with a strong focus on **data integrity, security, and healthcare relevance**, making it suitable as both a portfolio and academic project.

---

## üåç Project Motivation

With the growing role of technology in healthcare, HealthFlow aims to:
- Encourage proactive health management
- Centralize personal health data
- Lay a foundation for **future data analytics and AI-driven health insights**

---

##  Key Features

-  Secure user authentication (Register / Login / Logout)
-  Workout tracking (type, duration, intensity, calories burned)
-  Nutrition tracking (meals, calories, macronutrients)
-  Health metrics (weight, BP, sleep, hydration, mood)
-  Medication tracking and management
-  Daily dashboard health summary
-  Encrypted passwords & session-based access control
